# 記帳網站

## 功能
### 基本記帳功能
- 新增/編輯/刪除記帳記錄
- 自動計算總金額
- 支援店家/買家互斥輸入
- 必填欄位驗證機制

### 資料篩選功能
- 日期區間篩選
- 角色篩選（全部/店家/買家）
- 關鍵字搜尋
  - 店家/賣家名稱
  - 買家ID名稱
  - 商品名稱
  - 訂單類型
  - 訂單編號
  - 備註
### Excel 匯入匯出
- 支援 Excel 檔案匯入
- 匯出全部資料
- 匯出當前篩選資料
- 自動月份備份機制
- 重複資料智慧比對
### 資料視覺化 - 支援多種圖表類型
- 長條圖
- 堆疊長條圖（依訂單類型）
- 圓餅圖
- 折線圖
### 批量操作
- 批量選取記錄
- 批量刪除功能
- 全選/取消全選
### 資料安全相關
- 本地儲存 (LocalStorage)
- 自動備份機制
- 資料驗證保護

## 使用技術
- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js (圖表視覺化)
- SheetJS (Excel 處理)
- LocalStorage (本地儲存)

## 注意事項
- 資料儲存在瀏覽器的 LocalStorage 中
- 建議定期匯出備份重要資料
- 清除瀏覽器快取會導致資料遺失
- 請勿在無痕模式下使用
